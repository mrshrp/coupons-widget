Виджет купонов - Примеры использования

1. Автоматическая инициализация

Просто добавьте атрибут data-coupons-widget к любому элементу:

<div data-coupons-widget data-isin="RU000A0JXPG2"></div>

2. Программная инициализация

        <div id="my-widget"></div>

        <script>
        // После загрузки скрипта виджета
        const widget = new CouponsWidget({
          containerId: 'my-widget',
          data: [
            {
              paymentDate: '13.11.2025',
              rate: '25%',
              amount: '22.50₽',
              disabled: true
            },
            {
              paymentDate: '14.11.2025',
              rate: '25%',
              amount: '22.50₽',
              disabled: true
            },
            {
              paymentDate: '15.11.2025',
              rate: '25%',
              amount: '22.50₽',
              current: true
            },
            {
              paymentDate: '16.12.2025',
              rate: '25%',
              amount: '22.50₽'
            },
            {
              paymentDate: '17.01.2026',
              rate: '23%',
              amount: '21.15₽'
            }
          ]
        });
        </script>

3. Загрузка данных через API виджета

        <script>
          const widget = new CouponsWidget({
            containerId: 'moex-widget',
          });

          widget.load('RU000A106ZQ4');
        </script>

Виджет автоматически загружает данные с сайта Московской биржи
