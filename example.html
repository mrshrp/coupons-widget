Виджет купонов - Примеры использования

1. Автоматическая инициализация (светлая тема)

Просто добавьте атрибут data-coupons-widget к любому элементу:

<div data-coupons-widget data-isin="RU000A0JXPG2"></div>

4. Программная инициализация

        <div id="my-widget"></div>

        <script>
        // После загрузки скрипта виджета
        const widget = new CouponsWidget({
          containerId: 'my-widget',
          data: [
            {
              paymentDate: '15.11.2025',
              rate: '25%',
              amount: '22.50₽',
              current: true
            },
            {
              paymentDate: '15.12.2025',
              rate: '25%',
              amount: '22.50₽'
            }
          ]
        });
        </script>

5. Загрузка данных с MOEX по ISIN коду

        <!-- Автоматическая загрузка по ISIN -->
        <div data-coupons-widget data-isin="RU000A0JX0J2"></div>

        <!-- Программная загрузка -->
        <script>
          const widget = new CouponsWidget({
            containerId: 'moex-widget',
            isin: 'RU000A0JX0J2' // ISIN код облигации
          });

          // Или использовать метод load()
          widget.load('RU000A0JX0J2');
        </script>
